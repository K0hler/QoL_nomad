Справочник метрик — варианты компоновки

Как организовать определения внутри модального окна?

A

По секциям (как в карточке страны)

Метрики сгруппированы по секциям: Стоимость жизни, Качество жизни, Визы и т.д. Каждая секция — заголовок + список метрик с определениями. Консистентно с существующим UI.

Справочник показателей
💰 Стоимость жизни
Индекс стоимости жизни
Относительный показатель стоимости потребительских товаров. Нью-Йорк = 100. Ниже — дешевле.
Индекс аренды
Сравнительная стоимость аренды жилья. Нью-Йорк = 100. Ниже — дешевле.
🏥 Качество жизни
Индекс качества жизни
Комплексная оценка условий жизни. Выше — лучше.
B

Плоский список (алфавитный)

Все метрики в одном списке по алфавиту, без группировки. Проще для поиска конкретного термина, но теряется контекст секций.

Справочник показателей
Безопасность
Уровень безопасности в стране. Выше — безопаснее.
Загрязнение
Уровень загрязнения окружающей среды. Ниже — чище.
Здравоохранение
Качество и доступность медицины. Выше — лучше.
Индекс аренды
Сравнительная стоимость аренды. Нью-Йорк = 100.
